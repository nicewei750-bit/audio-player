# 本地小说音频播放器 - 安装指南

由于这是一个基于Web技术开发的PWA（Progressive Web App）应用，而非传统的原生iOS应用，因此安装方式与从App Store下载的应用有所不同。以下是将应用安装到iPhone上的详细步骤：

## 方法一：通过Web服务器访问并安装

### 步骤1：部署应用到Web服务器

1. 将应用文件上传到任何Web服务器（如GitHub Pages、Netlify、Vercel等）
2. 确保服务器支持HTTPS（iOS要求PWA必须通过HTTPS访问）
3. 获取应用的公开URL

### 步骤2：在iPhone上安装应用

1. 使用Safari浏览器访问应用的URL
2. 点击Safari底部的"分享"按钮（正方形带向上箭头的图标）
3. 在分享菜单中，向下滚动并点击"添加到主屏幕"选项
4. 为应用命名（例如"本地小说音频播放器"）
5. 点击"添加"按钮

现在，应用图标将出现在您的iPhone主屏幕上，就像普通应用一样可以点击打开。

## 方法二：本地服务器（适用于开发测试）

如果您想在本地测试应用，可以使用以下方法：

### 步骤1：在电脑上启动本地服务器

1. 确保您的电脑已安装Node.js
2. 在应用目录中打开终端
3. 运行以下命令安装并启动一个简单的HTTP服务器：

```bash
# 安装http-server（如果尚未安装）
npm install -g http-server

# 启动服务器
http-server -p 8080
```

4. 服务器启动后，您将看到类似以下输出：
   ```
   Starting up http-server, serving ./
   Available on:
     http://127.0.0.1:8080
     http://[您的IP地址]:8080
   ```

### 步骤2：在iPhone上访问并安装

1. 确保您的iPhone和电脑连接到同一个Wi-Fi网络
2. 在iPhone的Safari浏览器中输入您电脑的IP地址和端口号（例如：http://192.168.1.100:8080）
3. 点击Safari底部的"分享"按钮
4. 点击"添加到主屏幕"
5. 为应用命名并点击"添加"

## 方法三：使用ngrok（适用于远程访问）

如果您想从任何地方访问您的本地应用，可以使用ngrok：

1. 下载并安装ngrok（https://ngrok.com/）
2. 在应用目录中启动本地服务器（如方法二所述）
3. 在另一个终端窗口中运行：
   ```bash
   ngrok http 8080
   ```
4. ngrok将生成一个公开URL（例如：https://abc123.ngrok.io）
5. 在iPhone上使用Safari访问这个URL
6. 按照方法一的步骤3-5将应用添加到主屏幕

## 使用说明

安装完成后，您可以像使用普通应用一样使用"本地小说音频播放器"：

1. 点击主屏幕上的应用图标打开应用
2. 在"文件"标签页中点击"选择文件"或拖拽文件到应用中
3. 选择您想要播放的音频文件
4. 使用底部导航栏切换不同功能区域

## 注意事项

1. 由于iOS的限制，Web应用无法直接访问设备上的所有文件。您需要通过应用界面手动选择文件。
2. 应用数据（如播放列表）将存储在浏览器的本地存储中。如果您清除Safari的缓存，这些数据可能会丢失。
3. 对于最佳体验，建议使用iOS 13或更高版本。
4. 如果应用在添加到主屏幕后无法正常工作，请尝试删除并重新添加，或清除Safari的网站数据。

## 故障排除

### 应用无法安装到主屏幕
- 确保您使用的是Safari浏览器（iOS上只有Safari支持添加PWA到主屏幕）
- 确保网站通过HTTPS提供服务
- 检查网站是否正确配置了PWA所需的manifest.json文件

### 应用无法访问本地文件
- 确保您点击了"选择文件"按钮并授予了文件访问权限
- 尝试使用不同格式的音频文件（MP3、M4A等）

### 应用安装后无法启动
- 检查您的网络连接
- 确保服务器仍在运行（如果使用本地或远程服务器）
- 尝试删除并重新安装应用

如果您遇到任何其他问题，请随时联系开发者获取帮助。
